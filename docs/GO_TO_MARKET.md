# Neighbourhood Fit Score — Go-to-Market и монетизация

_Дата: 2026-06-01. Источник: 2 глубоких ресерч-агента (B2B-монетизация/GTM; consumer-клин + EU-расширение). Дополняет `COMPETITIVE_RESEARCH.md` и `IMPLEMENTATION_PLAN.md`._

## 0. Главная истина рынка

Consumer-оценки **бесплатны везде и всегда** — это SEO/воронка, не продукт. Деньги в категории — **B2B: API, white-label embed, per-report, data licensing**. Walk Score раздаёт score бесплатно, монетизирует API/виджет (от $115/мес), был куплен Redfin за **дистрибуцию виджета на 30k+ сайтах**, не за подписки. Local Logic — то же: бесплатно для consumer, embed на 8000 сайтах (22M MAU), монетизация — лицензия владельцу сайта (лесенка $20 → $250 → $500 → $1500). **Наш слой «как улучшить» + сценарии — то, что не продаёт никто.**

Размер рынка (сигнал): location intelligence ~$24–25B (2025), ~13–16% CAGR → $47B к 2030. PropTech ~$43–47B (2025) → ~$165B к 2035. Выходы на масштабе малой команды доказаны (Walk Score → Redfin; Local Logic $20.5M raised; UrbanFootprint $46M → куплен LightBox 2025).

## 1. Beachhead: с кого начинать (Brussels)

**#1 сегмент — сотрудники EU-институций и их семьи.** Почему именно они:

- **Размер и доступность:** Brussels-Capital ~1.25M жителей, **37.2% (464,629) не-бельгийцы**, ~287,590 граждан EU, ~200k+ экспатов, ~180 национальностей. Только Еврокомиссия — **~21,705 человек в Brussels**; +Совет, Парламент, агентства, NATO HQ (~4,000). Английский по умолчанию, высокая платёжеспособность.
- **Острота боли:** приезжают по дедлайну, не знают 19 коммун, решают commute-to-Schuman + двуязычные школы + катчмент. Голая карта не помогает — нужен ответ «для моего сценария».
- **Тёплый чокпоинт дистрибуции:** **Commission Welcome Office** (раздаёт гайд _«Soon in Brussels»_, листинги, проверку договоров) + **Expat Welcome Desk (be.brussels)** + commissioner.brussels онбордят почти всех. Один вход, куда fit-score встаёт нативно.
- **Churn:** новые когорты каждый цикл, stagiaires каждые 6 мес → постоянный приток.

Вторичные: NATO/int'l-org (тот же профиль). Корпоративные релоканты — **через релокейшн-агентства (B2B2C)**, не напрямую. Семьи/студенты — расширение, не старт (студенты — низкий WTP).

## 2. Что валидировать ДО тяжёлого билда

Ставка узкая: **меняет ли «почему + как улучшить, на сценарий» решение о районе** — против простого walkability-score (Strado) и ценового портала. Эксперименты по возрастанию усилия:

1. **Concierge MVP (≈0 €, дни).** Пост в r/Brussels + Brussels-expat FB-группах: «сделаю бесплатный персональный отчёт по коммуне — назови сценарий». Руками пишешь 10–20 отчётов. Меряет реальный спрос, ловит язык пользователей, проверяет — цитируют ли именно narrative+improve как решающее. **Начать здесь.**
2. **Landing + waitlist (~неделя).** «Найди свою коммуну Brussels по сценарию», 3 кнопки персон → email. A/B заголовка: _score-only_ vs _«почему + как улучшить»_. Дельта конверсии = первый сигнал, что нарратив важен. Трафик из #1 + €100–200 Meta/Reddit.
3. **Programmatic SEO commune-страницы (~2–4 нед, накопительно).** Шаблон на (коммуна × сценарий): «Etterbeek for remote workers», «Woluwe for families». ~19 коммун × 3 персоны = ~57 страниц (расширяемо до секторов). Ловит long-tail «[commune] for families», каждая со CTA на waitlist. **Это одновременно дистрибуция, демо и продакшн-пайплайн данных.**
4. **Embedded-пилот с одним каналом** (Welcome Office / релокейшн-агентство) — co-branded отчёт для их когорты. Высший сигнал, но медленно.

**Леаннейший решающий тест:** #1 → #2. Concierge говорит _меняет ли_ нарратив решение; landing A/B — _насколько_ нарратив поднимает intent над голым score.

## 3. Монетизация: лесенка (копируем Local Logic, ниже по цене для EU)

| Tier | Цена | Кому / что |
|---|---|---|
| **Free** | €0 | Consumer scenario-scores + «почему» на neighbourhood-страницах. SEO/воронка, никогда не платно. |
| **Agent / Pro** | **€19–29/мес** | Агенты/малые брокеры: безлимит брендированных отчётов + shareable listing-виджет. **Self-serve, карта — единственный покупатель без procurement.** |
| **Embed / SDK** | **€199–499/мес** | White-label scenario-виджет на портал/брокерский сайт, тарифицируется по views. |
| **API** | **от €750/мес** | Метрический доступ к scores + **«improve»-дельты** (уникальный payload). Free dev-tier ~1000 calls/день для посева. |
| **Improvement Report** | **€250–2500/отчёт** | Дифференциатор à la carte девелоперам/BID/советам (см. §4). |
| **Public-sector / consulting** | custom + **грант** | Scenario+equity дашборды городу, оплачено грантом, не бюджетом совета. |

Паттерн везде: free score → бесплатный embed-виджет (virality) → metered API → enterprise license/per-report. **Первый ICP за деньги: агенты/малые брокеры** (self-serve, мгновенно), затем **BID** для improvement-отчёта (автономный бюджет, быстро).

## 4. Слой «как улучшить» как продукт (незанятое)

Кто платит за «+1 аптека в 800 м → Senior 62→71»:

- **Девелоперы** — сильнейший коммерческий покупатель: amenity-gap = «что на первый этаж / какой участок лучше». Ложится на существующий feasibility-бюджет ($10k+). Продавать как **per-report**.
- **BID / Business Improvement Districts** — прямо покупают location-data, чтобы **обосновать инвестиции**; 61% управляются НКО с **автономным бюджетом** = быстро, без procurement. **Лучший ближайший платящий покупатель.**
- **Советы** — хотят обоснование капвложений, но медленный procurement → через **грант**, не продажу.
- **Агенты** — «район улучшается, вот закрывающийся разрыв» как listing-маркетинг (низкий WTP, в tier €19–29).

**Грант-трек (EU/Belgium), а не лицензия для public-sector:**
- **European Urban Initiative (EUI) – Innovative Actions:** 4-й call, **€60M ERDF, до €2M/проект**, города >25k. Brussels имеет EUI/URBACT NCP и активные проекты.
- **Horizon Europe «100 Climate-Neutral & Smart Cities»** — Brussels ровно профиль.
- **smartcity.brussels / Living-in.EU / OASC** — региональный civic-tech.
- **Прецедент:** сам Walk Score стартовал на грантах **Rockefeller + Robert Wood Johnson Foundation** — public-good версия этого продукта имеет доказанный грант-трек.

## 5. GTM-клин для соло-строителя

Что инкумбенты делали в начале: **дистрибуция била данные.** Walk Score раздал free score + **copy-paste embed-виджет** → 30k+ сайтов → moat → Redfin. Local Logic пошёл через **channel-partners** (MoxiCloud, франшизы Century 21/RE/MAX/Sotheby's), не прямые продажи.

**Рекомендуемая последовательность:**
1. **Programmatic SEO neighbourhood-страницы (Brussels).** Порталы слабы в neighbourhood-нюансе — локальный игрок выигрывает long-tail. Сотни страниц (коммуна × сценарий) со score + «почему» + «улучшить» = **бесплатная воронка И демо И пайплайн** одновременно.
2. **Бесплатный embed scenario-виджет** (ход Walk Score) для EU-агентов — copy-paste, брендированный, ссылка назад. Сеет install-base под будущий paid embed/API.
3. **Первая выручка с агентов/малых брокеров (€19–29/мес)** — единственный полностью self-serve покупатель.
4. **Монетизировать дифференциатор через BID (кэш сейчас) + советы через гранты (параллельно).**
5. **v2-триггер:** при заметном install-base виджета + одном платном improve-отчёте — поднимать API/embed в порталы через **channel-partner** (ход Local Logic/MoxiCloud), не прямые enterprise-продажи.

**Жёсткие правды:** страховщики/кредиторы и enterprise retail/site-selection — **недостижимы соло** (планка data-provenance и procurement). Релокейшн/HR — только как **поставщик данных в RMC**, не напрямую. Советы — грант, не лицензия. Первые 6–12 мес: **агенты (кэш) + BID/девелоперы (improve-отчёт) + SEO/виджет (дистрибуция)**, с грант-заявкой параллельно.

## 7. Юридические гайдрейлы (важно с первого дня)

Риск реален, но управляем — это defamation/property-value и оптика дискриминации, **не жёсткий EU-запрет**.

1. **Оцениваем СРЕДУ (amenities, транспорт, зелень, сервисы), НИКОГДА не жителей.** Привязка score к тому, _кто_ живёт — политически радиоактивна (кейс Rotterdam Act / Wbmgp, отзыв Art. 8 в 2022 на фоне обвинений в дискриминации).
2. **Источник — объективные открытые данные, не мнение.** Атрибутировать каждое число (Statbel, Leefbaarometer, OSM, STIB/TfL). Так делают Zoopla («third-party data providers») и AreaVibes (публикует методологию + источники).
3. **«As is / no warranty»** терминология (модель Rightmove) — информация/decision-support, не совет.
4. **Относительно и конструктивно, не уничижительно.** Leefbaarometer репортит как «отклонение от среднего», не «плохо/хорошо». **Наш «как улучшить» — юридический актив:** переформулирует низкий score в _изменяемый контекст_, а не приговор району.
5. **Persona-relative, не абсолютно.** «Ниже fit _для семейного сценария_ из-за разрывов X» безопаснее, чем «плохой район».
6. **GDPR:** только агрегат на уровне сектора, никаких индивидуальных данных.

## 8. Доказательства спроса + SEO-клин (детально)

### 8.1 Спрос подтверждён (косвенно, но убедительно)

- **Рынок большой/повторяющийся/сезонный.** EU-институции ~79,211 сотрудников (2024); Blue Book стажёры ~1,900/год, **фиксированные интейки 1 марта / 1 октября** (поиск пиком авг-сен и янв-фев); ~37% не-бельгийцев в Brussels. Высокий-intent, high-LTV, без локального знания.
- **Search-спрос виден по плотности SERP:** по «where to live in Brussels» конкурируют **15+ доменов** (Expatica, HousingAnywhere, Spotahome, Cohabs, Colive, Expat.com…) — никто не пишет контент под keyword с нулём volume. Long-tail «[commune] × scenario» — 10–150/мес каждый, индивидуально мал, коллективно значим, **никем не таргетится**.
- **Форумы/комьюнити:** ежегодный «Best commune in Brussels» (Brussels Notes Substack, краудсорс «30 минут за обедом, без методологии»), повторяющиеся треды Expat.com/r/Brussels. Точный keyword-volume — **единственное число проверить тулом (Ahrefs/Semrush) ДО билда**.
- **Сезонность = актив:** общий relocation-пик май-авг (сдвиг в раннюю весну); + стажёрский pulse дважды в год. → Страницы должны ранжироваться к **Q4 2026 / янв 2027**, чтобы поймать весну-2027.

### 8.2 Угроза: Google «scaled content abuse» (март 2024)

Наивный programmatic («[scenario] in [commune]» × 500 с подстановкой) **убивается** обновлением (плюс может просадить весь домен). Выживание = **реальные данные на странице**: уникальный score + реальные POI + нарратив «почему» из data-дельт vs медиана Brussels. Это и есть ров — лень-конкуренты выпиливаются, награждается единственный инструмент с настоящими данными.

### 8.3 Гранулярность и шаблон

- **Коммуны сначала (все 19), кварталы потом (НЕ все 120).** Люди говорят «живу в Ixelles», не «в quartier Châtelain». 120×3 взорвёт матрицу и триггернёт scaled-content на новом домене. Топ-кварталы (Châtelain/Flagey/Sablon/EU Quarter) — фаза 2.
- **URL:** `/brussels/{commune}/{scenario}/`. Обязательный уникальный контент на странице: numeric score, 6+ sub-scores с реальными значениями (rent, crime, школы, % зелени, commute-to-Schuman, % иностранцев), названные реальные POI, нарратив «почему» из дельт, scenario-specific «как улучшить».
- **Schema.org:** `Place` + `Dataset` + `FAQPage` + `BreadcrumbList` (JSON-LD). **НЕ `LocalBusiness`** (type mismatch). Subfolders, не subdomains. Noindex страницы без достаточных данных.

### 8.4 Виджет (ход Walk Score) — чисто по гайдлайнам

Copy-paste бейдж «Etterbeek — Family Fit 78/100 · powered by NFS», агент встраивает на листинг (помогает сдать интернациональным арендаторам). Backlink **должен быть editorially-earned**: виджет несёт самостоятельную ценность, ссылка опциональна/`nofollow`/attributional «powered by», не обязательна, не exact-match anchor. Трактовать виджет как **канал дистрибуции+бренда**, который *заодно* сеет чистые ссылки — не как main link-building.

### 8.5 Тайминг и leanest-валидация

Новый домен: индексация дни-2нед, первые long-tail ранги нед 3-6, **значимый трафик 6-12 мес**. **НЕ запускать всю матрицу** — сначала **5 hero-страниц** (Etterbeek/family, Ixelles/remote, Woluwe-SP/senior, compare Etterbeek-vs-Ixelles, landing /family/), hand-verify данные. **Метрика валидации: ≥1 hero-страница в топ-20 по своему long-tail за 90 дней + ≥1 чистый backlink виджета.** Потом scale до ~57 (19 коммун × 3 + comparisons). Инструментировать воронку (view → fit-checker → widget-copy/email).

## Резюме (две рекомендации)

1. **Beachhead + первый тест:** сотрудники EU-институций и семьи (~50k+, Commission ~21,705) — острейшая боль «какая из 19 коммун», высокий WTP, churn, тёплый чокпоинт (Welcome Office + Expat Welcome Desk). Первый эксперимент — **бесплатный concierge «commune fit report»** в r/Brussels и expat-FB, затем **landing + waitlist A/B «score-only» vs «почему + улучшить»**. Валидирует главное за ~0 €.
2. **Второй город — Amsterdam/NL:** выигрывает по всем трём осям (лучшие открытые данные через Leefbaarometer, разрыв нетронут, высокий English-default expat-спрос) + полное переиспользование OSM из Brussels-пайплайна. Berlin — #3, London — самый data-rich, если ослабить «EU-first».
